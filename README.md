# campus-resource-frontend

校园资源管理系统前端项目

## 技术栈

- Vue 3
- Vite
- Element Plus
- Vue Router
- Pinia
- ECharts
- Axios

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

## 项目结构

```
src/
├── api/          # API接口
├── assets/       # 静态资源
├── components/   # 组件
├── router/       # 路由配置
├── store/        # 状态管理
├── utils/        # 工具函数
└── views/        # 页面视图
```
